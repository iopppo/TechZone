ОПИСАНИЕ ПРОЕКТА:
----------------
Сайт компьютерного магазина комплектующих "TechZone".
Состоит из трёх страниц:
1. Главная (index.html) - приветствие, преимущества, популярные товары
2. Каталог (catalog.html) - информационная страница с товарами
3. Контакты (contact.html) - форма обратной связи и контактная информация

ИСПОЛЬЗОВАННЫЕ ТЕХНОЛОГИИ:
--------------------------
- HTML5
- CSS3 (Flexbox, Grid)
- Bootstrap 5
- Font Awesome иконки
- CSS-анимации

ОСОБЕННОСТИ ДИЗАЙНА:
--------------------
- Тёмная тема с неоновыми акцентами
- Адаптивный дизайн (мобильные, планшеты, десктоп)
- Плавные анимации и переходы
- Эффект свечения при наведении
- Градиентные элементы
- Стеклянный эффект (backdrop-filter)

СТРУКТУРА ПРОЕКТА:
------------------
index.html          - Главная страница
catalog.html        - Каталог товаров
contact.html        - Контакты и форма обратной связи
style.css           - Дополнительные стили
bootstrap.min.css   - Bootstrap фреймворк
